## Detailed Setup (30 mins)

### For Agencies / Freelancers Selling This:

### Step 1: Prepare the Content Pack

#### 1.1 Customize for Your Client
Replace all instances of:
- `Aura Skin & Hair Studio` → `[Client Name]`
- `Vijayawada` → `[Client City]`
- `+91-XXXX-XXXX-XX` → `[Client Phone]`
- `hello@aurastudio.in` → `[Client Email]`

**Tool:** Use "Find & Replace" in any text editor (Ctrl+H / Cmd+H)

#### 1.2 Update Prices
Open all blog files and update pricing to match client's rates.

**Example in pillar blog:**
```
Before: "Price Range: ₹1,500-₹4,500"
After: "Price Range: ₹2,000-₹5,000"
```

#### 1.3 Personalize Services
Update service list to match what client actually offers:
- Remove services they don't provide
- Add unique services they do offer
- Update descriptions

### Step 2: Set Up Website

#### Option A: WordPress (Recommended)
1. Install WordPress on hosting
2. Install theme (e.g., Neve, GeneratePress)
3. Create new blog posts from markdown files
4. Set featured images for each post
5. Customize navigation links

#### Option B: Use Provided HTML
1. Download `website-index.html`
2. Open in any web browser
3. Looks great immediately
4. Deploy to any hosting (Netlify, Vercel, GitHub Pages - all free)

**Steps to deploy on Netlify (free, takes 5 mins):**
- Go to netlify.com
- Drag & drop `website-index.html` file
- Done! Get shareable link instantly

#### Option C: Squarespace / Wix
1. Create new blog post for each markdown file
2. Copy text content (without markdown formatting)
3. Format headings (H1, H2, H3)
4. Add links between articles
5. Set SEO title & description for each post

### Step 3: Internal Linking

After publishing all blogs, go through each one and:

1. Find placeholder links like `[link-to-keratin-blog]`
2. Replace with actual URL:
   - Replace `[link-to-keratin-blog]` with `/blogs/keratin-treatment` (or actual URL)
   - Replace `[link-to-bridal-blog]` with `/blogs/bridal-makeup`
   - Replace `[link-to-skincare-blog]` with `/blogs/skincare-facials`

3. Also create reciprocal links:
   - In pillar blog, link to all 5 supporting blogs
   - In each supporting blog, link back to pillar blog

### Step 4: Meta Descriptions

For SEO, add meta descriptions to each blog:

**Example:**
```
Pillar Blog Meta: "Best salon in Vijayawada for hair treatments, facials, and bridal packages. Professional services, certified experts, transparent pricing. Book now!"

Keratin Blog Meta: "Keratin treatment for hair explained. Cost, duration, aftercare, and why it works in Vijayawada's climate. See results in 3 days!"

Bridal Blog Meta: "6-week pre-bridal package guide. Skin prep, hair treatments, makeup trial, and bridal day services. Start your glow-up for the big day!"
```

**How to add:**
- WordPress: SEO plugin (Yoast, All in One SEO)
- HTML: `<meta name="description" content="...">`
- Wix/Squarespace: Built-in SEO settings

### Step 5: Publish Schedule

**Week 1:** Pillar blog only
- Publish pillar blog
- Optimize for search engines
- Get indexed (3-7 days)

**Week 2:** Supporting blogs #1 & #2
- Publish Hair Services blog
- Publish Skincare blog
- Link from pillar blog

**Week 3:** Supporting blogs #3 & #4
- Publish Bridal blog
- Publish Keratin blog

**Week 4:** Supporting blog #5
- Publish Local Optimization blog

**Why spread out?** Google favors consistent content over dumps. Spread = better ranking.

## Google My Business Setup

**Why important:** 90% of salon searches use Google Maps/GMB.

### Step 1: Claim Your Listing
1. Go to google.com/business
2. Sign in with your Google account
3. Search for your salon name
4. Click "Claim this business"
5. Verify via postcard (or phone if available)

### Step 2: Complete Your Profile
- ✅ Business name
- ✅ Full address
- ✅ Hours of operation
- ✅ Phone number
- ✅ Website URL
- ✅ Service categories (Hair salon, Skincare, Bridal)
- ✅ 10+ high-quality photos
- ✅ Detailed service list with prices

### Step 3: Add Services & Pricing
1. In GMB dashboard, go to "Services"
2. Add each service:
   - Service name (e.g., "Keratin Treatment")
   - Category (Hair Care)
   - Price
   - Duration
   - Description

### Step 4: Encourage Reviews
- Ask happy customers to review on Google
- Respond to all reviews (positive & negative)
- Target: 50+ reviews, 4.5+ rating

## Tracking & Analytics

### Google Search Console Setup

**Why:** See how your blogs rank, what keywords drive traffic, fix errors.

1. Go to search.google.com/search-console
2. Add your website
3. Verify ownership (DNS record or HTML file)
4. Submit sitemaps (usually automatic if WordPress)
5. Check "Performance" tab after 2-4 weeks to see:
   - Impressions (how often you show up in search)
   - Clicks (how often people visit)
   - Average ranking position
   - Top keywords

### Google Analytics Setup

**Why:** Track visitor behavior, conversion rate, bounce rate.

1. Go to analytics.google.com
2. Create new property for your website
3. Install tracking code (WordPress: plugin like MonsterInsights)
4. Wait 24-48 hours for data
5. Check:
   - Total visitors per month
   - Traffic source (organic = Google search)
   - Pages with highest traffic
   - Conversion goal (bookings, calls, form fills)

## Troubleshooting

### Issue 1: Blogs Not Appearing in Google Search

**Why:** Google needs 1-2 weeks to index new pages.

**Solution:**
1. Submit to Google Search Console manually
2. Wait 7-14 days
3. Search for your exact blog title: `"best salon in vijayawada"`
4. If still not appearing after 2 weeks, check robots.txt isn't blocking indexing

### Issue 2: Poor Rankings (Position 50+)

**Why:** Takes 3-6 months to rank, competition may be high.

**Solution:**
1. Update blog with more internal links
2. Improve on-page SEO: add more keyword mentions, better headings
3. Build backlinks: guest posts on other blogs, local directory listings
4. Add more content (publish more blogs)
5. Improve user experience: faster page load, mobile-friendly

### Issue 3: No Traffic Despite Good Rankings

**Why:** Title/description not compelling to click, or rankings aren't visible in top 3.

**Solution:**
1. Improve meta description to increase click-through rate (CTR)
2. Rank higher: build backlinks, update content quality
3. Check Google Search Console: see which keywords show your page but don't get clicks
4. Rewrite titles/descriptions to be more compelling

### Issue 4: Website is Slow

**Why:** Images too large, poor hosting, too many plugins.

**Solution:**
1. Compress images: use Tinify.com (free)
2. Use quality hosting: Bluehost, SiteGround, Vercel
3. Remove unnecessary plugins
4. Use caching plugin: WP Super Cache (WordPress)

### Issue 5: Not Getting Leads from Blog

**Why:** No clear call-to-action, visitors can't figure out how to book.

**Solution:**
1. Add "Book Now" button on every blog (make it prominent/red)
2. Simplify booking process: direct link to scheduling tool
3. Add phone number prominently
4. Add WhatsApp button
5. Create contact form
6. Track where leads come from (Google Analytics)
