# 📝 PILLAR BLOG PROMPT

**Use this prompt to generate the main authority blog that targets your primary keyword.**

---

## 🎯 How to Use This Prompt

1. **Copy the entire prompt below** (everything between the dashes)
2. **Replace placeholders:**
   - `[BUSINESS_NAME]` → Your salon/clinic name
   - `[CITY]` → Your city
   - `[SERVICE_TYPE]` → salon, skincare clinic, dental clinic, etc.
   - `[PRIMARY_KEYWORD]` → Main SEO keyword (e.g., "best salon in Vijayawada")
   - `[SERVICES_LIST]` → What you offer (hair, skin, nails, bridal, etc.)
   - `[CTA_GOAL]` → What action you want (book appointment, call us, contact form)

3. **Paste into Claude / ChatGPT / Gemini**
4. **Get a full ~1800-word blog post**
5. **Customize with images, testimonials, your specific prices**

---

## ✂️ THE PROMPT (Copy Everything Below)

```
You are an expert SEO content strategist for local service businesses.

TASK: Generate a comprehensive pillar blog post (1800-2000 words) 
that ranks for the keyword "[PRIMARY_KEYWORD]"

BUSINESS INFO:
- Business Name: [BUSINESS_NAME]
- Location: [CITY]
- Type: [SERVICE_TYPE]
- Services: [SERVICES_LIST]
- Primary Goal: [CTA_GOAL]

KEYWORD TARGET: "[PRIMARY_KEYWORD]"
SEARCH INTENT: Local users searching for the best [SERVICE_TYPE] in [CITY]

REQUIREMENTS:
1. Structure MUST include:
   - H1 (includes keyword): e.g., "Best [Service] in [City] – Services, Cost & What to Expect"
   - H2 sections (4-5): Introduction, Services Offered, Why Choose Us, Cost/Pricing, How to Book
   - H3 subsections (2-3 per H2 section)
   - Bold keywords naturally throughout
   - 1-2 real statistics/data points

2. Content MUST:
   - Be beginner-friendly but authoritative
   - Include local references to [CITY] (3-4 mentions naturally)
   - Address customer pain points
   - Include 2-3 call-to-action buttons (e.g., "Book Now", "Call Us")
   - End with FAQ section (5-7 common questions)
   - Internal link suggestions in square brackets [link-to-supporting-blog]
   - Mention specific services offered by [BUSINESS_NAME]

3. Tone: Professional but friendly, trustworthy, expert
4. Format: Markdown with proper heading hierarchy
5. Length: 1800-2000 words minimum

INTERNAL LINKING OPPORTUNITIES:
- Link to "hair treatment guide" blog
- Link to "skincare services" blog  
- Link to "bridal makeup packages" blog
- Link to "pricing guide" blog

OUTPUT FORMAT:
```markdown
# [H1 HEADING WITH KEYWORD]

## Introduction
[Engaging opening about why choosing right salon matters in Vijayawada...]

## [H2 SECTION 1]
### [H3 subsection]
[Content...]

## [H2 SECTION 2]
[Detailed content with benefits]

## Frequently Asked Questions
**Q: Common question?**
A: Answer with link to supporting blog if relevant...

## Ready to Book?
[CTA: Contact information and booking link]
```

TONE & STYLE:
- Use "we" to refer to the business
- Write in second person ("you") for audience
- Include one personal story/client testimonial
- Use power words: "expert," "trusted," "proven," "personalized"
- Break up text with short sentences after long ones
- Use lists and bullet points to improve readability

KEYWORD PLACEMENT:
- H1: Include primary keyword naturally
- First 100 words: Include keyword 2x
- Throughout: Include keyword 3-4 more times (2-3% keyword density)
- Variations: "best salon in Vijayawada", "top-rated salon", "professional salon services"

SEO BEST PRACTICES:
✓ Meta description (160 chars): Write a compelling meta that includes keyword
✓ Include statistics (e.g., "85% of clients see results in 2 visits")
✓ Address competitor blogs without naming them
✓ Include local schema hints (city name, service types, pricing ranges)
✓ End with strong call-to-action

NOW WRITE THE COMPLETE BLOG POST:
```
